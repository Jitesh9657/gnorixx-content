## About GnorixX

GnorixX is a suite of AI-powered script tools for short-form content creators — Reels, TikTok, and YouTube Shorts.

- [Idea Generator](https://gnorixx.com/tools/idea-generator) — get new ideas with untapped potential angles and sub-niches
- [Script Engine](https://gnorixx.com/tools/script-engine) — turn ideas into ready-to-record scripts
- [Script Enhancer](https://gnorixx.com/tools/review/short-form) — diagnose and rewrite weak scripts
- [YT Keywords](https://gnorixx.com/tools/yt-keywords) — keyword research for YouTube
